# opencv_reconhecimento
opencv - instalação e scripts de reconhecimento.

Primeiro passo é a instação do ambiente open cv em sua máquina linux. Fiz para o Ubuntu 16.04 seguindo o tutorial:
https://www.learnopencv.com/install-opencv3-on-ubuntu/

Após a instalação efetue os teste propostos no site para o teste do ambiente.
